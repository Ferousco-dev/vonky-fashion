## Overview

Vonky Fashion is a single-page brand experience built around motion. It features a preloader, animated page transitions, scroll-triggered reveals, and an editorial layout that showcases the collection with a high-fashion aesthetic.

## Features

- 🎬 Scroll-triggered animations with GSAP + ScrollTrigger
- ⏳ Custom preloader and page-transition effects
- 🍔 Responsive navigation with hamburger menu
- 🖼️ Editorial hero, about, and collection sections
- 🎨 Playfair Display typography and Font Awesome icons

## Tech Stack

- **Markup:** HTML5
- **Styling:** CSS3
- **Animation:** GSAP + ScrollTrigger
- **Scripting:** Vanilla JavaScript

## Getting Started

```bash
git clone https://github.com/Ferousco-dev/vonky-fashion.git
cd vonky-fashion
# open index.html in your browser, or serve it:
```
